O projeto consiste em um cadastramento de tarefas.
Foi realizado um CRUD (Create, Read, Update, Delet) com o Banco de Dados PostgreSQL.
Foi implementado duas views:
1a - Tela de Cadastro de Tarefas;
2a - Tela de Listagem de Tarefas.
Na segunda tela podemos listar, concluir, excluir, atualizar e buscar tarefas.

Implementações:

- Aplicação Java;
- Padrão MVC (model, view, controller);
- Iteração com banco de dados.

OBSERVAÇÃO: 
O arquivo está sendo implementado JSF utilizando JPA e persistencia no PostgreSQL.
